El valor esperado te dice si un precio es rentable a largo plazo, no si un boleto individual ganara hoy.

## Formula Base de +EV

Usa esta formula:

- EV = (Probabilidad de Ganar x Ganancia si Acierta) - (Probabilidad de Perder x Monto Apostado)

Donde:

- Probabilidad de ganar es tu proyeccion propia convertida a decimal.
- Probabilidad de perder = 1 - probabilidad de ganar.
- Ganancia si acierta depende de la cuota y del monto apostado.

## Probabilidad de Equilibrio

Cada cuota tiene una tasa minima de acierto para no perder valor.

- Equilibrio % = (1 / Cuota Decimal) x 100

Si tu proyeccion supera ese nivel, tienes edge positivo.

## Ejemplo Rapido

- Monto: $100
- Cuota de mercado: 2.10 decimal
- Tu probabilidad proyectada: 52%

Calculo:

- Ganancia si acierta = $110
- EV = (0.52 x 110) - (0.48 x 100) = 57.2 - 48 = +$9.20
- EV% = 9.20%

Ese escenario tiene valor esperado positivo.

## Flujo Practico

1. Convierte la cuota a probabilidad implicita y equilibrio.
2. Estima tu probabilidad real.
3. Calcula EV y edge.
4. Solo considera jugadas con EV positivo claro.
