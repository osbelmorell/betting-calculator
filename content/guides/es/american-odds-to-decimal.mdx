Las líneas americanas parecen complejas al inicio, pero la conversión es directa si separas líneas positivas y negativas.

## Fórmula para Líneas Positivas

Usa esta fórmula:

- Decimal = (Americana / 100) + 1

Ejemplo con +150:

- Decimal = (150 / 100) + 1 = 2.50

## Fórmula para Líneas Negativas

Usa esta fórmula:

- Decimal = (100 / |Americana|) + 1

Ejemplo con -110:

- Decimal = (100 / 110) + 1 = 1.909

## Convertir Decimal a Fraccional

Cuando ya tienes la línea decimal:

- Fraccional = Decimal - 1

Ejemplo con 2.50 decimal:

- Fraccional = 1.50 = 3/2
