# GA Reports Folder

Use this folder to store Google Analytics report snapshots shared during optimization cycles.

Naming convention:
- `YYYY-MM-DD-<scope>-ga-report.md`
- Example: `2026-03-28-example-funnel-ga-report.md`

Minimum sections expected in each report file:
1. Date range and timezone
2. Property name and stream
3. Funnel definition (events and order)
4. Segments used
5. Raw metrics and conversion rates
6. Notes on anomalies
7. Questions for next iteration
